    FastAPI 0.1.0¶

    Dataset Recommendation Service¶

    GET /dataset-recsys/recommend¶

    Get recommendations

    Description

    Retrieve the top-N recommendations for a given item in a dataset.

    The meaning of item varies by dataset: - MathE â€” items are educational materials (PDFs), whose identifiers correspond to filenames with .pdf extension (e.g., {item_id}.pdf).

    Input parameters

    Parameter In Type Default Nullable Description
    dataset query string No The dataset/application name
    iid query string No The item identifier within the selected dataset
    n query integer 10 No Number of similar items to return

    Response 200 OK 

    {
    "dataset": "mathe",
    "iid": "56.pdf",
    "recommendations": [
        "54.pdf",
        "55.pdf",
        "52.pdf",
        "367.pdf",
        "360.pdf"
    ]
}
